# HexSoftwares Resume Website

A personal resume website built using HTML and CSS as part of the Hex Software Internship Program.

## Project Description
This is an interactive, responsive resume website showcasing my professional experience, skills, projects, and contact information. The website features a modern design with smooth animations and is fully mobile-responsive.

## Features
- Professional header with profile picture and social media links
- About Me section with professional summary
- Skills showcase with hover effects
- Projects portfolio section
- Work experience timeline
- Contact information
- Downloadable PDF resume
- Smooth scrolling and animations
- Mobile-responsive design

## Technologies Used
- HTML5
- CSS3
- Responsive Design
